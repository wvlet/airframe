airframe-bootstrap
====

airframe-bootstrap is a library for helping application bootstrap process with airframe-config and airframe DI. 

## Usage

[![Maven Central](https://maven-badges.herokuapp.com/maven-central/org.wvlet.airframe/airframe-codec_2.12/badge.svg)](https://maven-badges.herokuapp.com/maven-central/org.wvlet.airframe/airframe-codec_2.12/)

```scala
libraryDependencies += "org.wvlet.airframe" %% "airframe-bootstrap" % "(version)"
```
