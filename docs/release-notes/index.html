22.4.1

This version upgrades to Scala.js 1.10.0 to address a security vulnerability CVE-2022-28355 found in java.util.UUID.randomUUID() implementation of Scala.js. If UUID.randomUUID() is used for generting publicly visible IDs, generated IDs were preditable in the former Scala.js versions.
Similarily, airframe-ulid for Scala.js is fixed to use SecureRandom to avoid generating predictable ULID values.

  • Upgrade to Scala.js 1.10.0 (#2122) [d2ce4725]
  • airframe-ulid: Use SecureRandom-based generator for Scala.js (#2121) [07936e55]
  • Add scala-js-java-securerandom for Scala.js 1.10.0 (#2117) [96f5d37e]
